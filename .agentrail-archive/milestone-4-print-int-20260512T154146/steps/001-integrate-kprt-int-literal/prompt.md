Extend KPRT in snobol4/src/emit_asm.sno with an integer-literal dispatch path. The pattern matches PRINT * <comma+space> <digit-run>, captures NUM, and emits la r0,N; push r0; jal r1,(_putint); add sp,3 + a trailing newline via _putc(10). Splice _putint runtime from snobol4/runtime/putint.s via awk in scripts/fortran (dcsno static-program-size-limit forces this split; see tools/briefs/dcsno-static-program-size-limit.md). Demo: examples/print-int.f -> '42'. Regression: examples/hello.f still produces 'Hello, World!'.

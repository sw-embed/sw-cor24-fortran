Add KDO and KCNT to snobol4/src/emit_asm.sno. KDO parses 'DO <lbl> <var> = <start>, <end>' (integer literal start/end only), emits the init (la start; la _V_<var>; sw) and a fresh loop-top label LDT<n>; saves DOLBL/DOVAR/DOEND/DOTOP state. KCNT checks if the source-label LBL matches saved DOLBL; if yes, emits increment + bound-test + brt-back-to-DOTOP (cls r0,r1 where r1=end+1). Cap end+1 computed at compile time in SNOBOL4 via DOEND1 = DOEND + 1. Demo: examples/sum10.f -> '55'. Regress all 5 prior demos.
